# Aufzuege!

Aufzuege! is a Foundry VTT module for configurable elevator style movement and scene transition workflows.

## Features

- Elevator panel style interface for travel choices
- Configurable destinations and transition handling
- Localized strings for German and English
- Styled templates for panel and configuration UI
- Optional elevator sound effects

## Installation

1. Open **Add-on Modules** in Foundry VTT.
2. Click **Install Module**.
3. Paste this manifest URL:
   `https://github.com/LisaNeoGermany/aufzuege/releases/latest/download/module.json`
4. Click **Install**.

## Compatibility

- Minimum Foundry version: 12
- Verified Foundry version: 14
